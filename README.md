# 信披-定向知识获取

面向公募基金**定期报告（信披文档）**的定向信息提取工具 demo，使用 AI 从 PDF 定期报告中自动抽取指定章节内容，降低人工检索与汇总成本。

## 功能模块

| 模块 | 说明 |
|---|---|
| 任务管理 | 信息提取任务的统一管理：查看执行状态/进度/记录、手动发起、定时调度 |
| 结果筛选与展示 | 多条件筛选提取结果，表格化呈现，支持导出 / 删除 |

## 文件清单

| 文件 | 说明 |
|---|---|
| `index.html` | 页面主文件：整体结构与内容布局 |
| `css/style.css` | 页面业务样式（筛选区、表格区、布局） |
| `css/component.css` | 公共组件样式（顶栏、页签、按钮、分页等） |
| `js/data.js` | Mock 数据：演示用模拟数据 |
| `js/app.js` | 页面交互逻辑 |
| `需求说明.md` | 需求背景、页面功能、目录结构规划 |

## 本地预览

该项目为纯静态文件，无需启动服务。两种打开方式：

### 方式一：直接打开（最简单）

双击 `index.html`，默认浏览器即可加载。

### 方式二：本地静态服务（推荐）

部分浏览器对 `file://` 协议下的某些 API 有安全限制，建议用静态服务器：

```bash
# 进入项目目录后任选其一
python -m http.server 8000
# 或
npx serve .
```

随后访问 <http://localhost:8000/index.html>。
